# <center>android apk reverse tools</center>

1. [apktool](https://apktool.org/) - Reverse engineering Android apk files

2. [jd-gui](https://java-decompiler.github.io/) - Displays Java source codes of “.class” files

3. [dex2jar](https://github.com/pxb1988/dex2jar/releases) - Tools to work with android .dex and java .class files
